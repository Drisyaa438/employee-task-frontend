# Employee Task Management - Frontend (React)

This is the **frontend admin panel** for the Employee Task Assignment & Tracking System.

Built using **React.js** with REST API integration to the Django backend.  
Deployed on **Vercel** for production hosting.

---

Features

✅ JWT-based secure login (Admin & Employee roles)  
✅ Role-based dashboards  
✅ Admin can create, assign, and monitor tasks  
✅ Employees can view and complete assigned tasks  
✅ Dynamic, responsive UI using Tailwind CSS  
✅ Protected routes using React Router  

---

Tech Stack

- **Frontend:** React.js (CRA)
- **Styling:** Tailwind CSS
- **State Management:** React Hooks
- **Authentication:** JWT Tokens (from backend)
- **Build Tool:** npm
- **Hosting:** Vercel

---

Backend API Integration

This app connects with the Django REST API at:
http://127.0.0.1:8000/api/


Make sure your backend is running before testing locally.

Running Locally

npm install
npm start

To build for production:
npm run build
